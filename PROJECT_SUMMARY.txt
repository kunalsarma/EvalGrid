================================================================================
                          QUICK START GUIDE
================================================================================

1. INSTALL DEPENDENCIES
   cd /path/to/EvalGrid
   npm install

2. START DEVELOPMENT SERVER
   npm run dev
   â†’ Opens at http://localhost:5173/

3. PREPARE YOUR DATA
   - Click "Download Template"
   - Add your data to the Excel file
   - Save and upload

4. CONFIGURE EVALUATION
   - Paste Google AI Studio API key
   - Select Gemini model and tier
   - Add evaluation fields with:
     * Field name
     * Field description
     * Target column
     * Context columns
     * Score definitions

5. RUN EVALUATION
   - Click "Start Evaluation"
   - Monitor real-time progress
   - Download results when complete

6. BUILD FOR PRODUCTION
   npm run build
   â†’ Creates optimized dist/ folder
